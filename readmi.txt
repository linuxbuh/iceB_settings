Описание настроечных файлов для подсистемы iceB с терминальным интерфейсом

xterm         - файл для программы "gnome-terminal".Стандарный файл xterm не подходит так как не все нажатия
                клавиш на клавиатуре обрабатываются правильно. Стандарный файл подходит только для
                программы "xterm". Если вы замените стандартный файл на новый, то тогда в программе 
                "xterm" не все клавиши будут обрабатываться правильно а в "gnome-terminal" правильно.
                Файл нужно ложить туда где находится системный xterm. Обычно это каталог /lib/terminfo/x
                
		В разных дистрибутивах это может быть разный каталог.
		Для корректной работы программ iceB с этим файлом они должны
                быть построены с параметром MAXFKEYS=10 (смотрите файл
                iceb.h в корне папки где находится система)
		
		В debian 8 родной файл xterm подходит для работы с 
		программами системы iceB в консоли "gnome-terminal" если она построена с параметром
		MAXFKEYS=12 Вероятнее всего и в следующих версиях debian
		будет этот же файл.

		Для того чтобы узнать с каким параметром MAXFKEYS построены
		программы системы iceB, нужно запустить программу i_admint и
		выбрать в меню "Коды функциональныйх клавиш применяемые в
		программе". После нажатия клавиши "Enter" на экране появится
		список функциональных клавиш с их кодами а в первой строке
		будет рапечатано значение параметра MAXFKEYS
		
xterm-putty   - файл для программы "putty". С помощью программы "putty" можно зайти на линуксовую машину
                в режиме консоли из под Windows и работать с iceB. Файл должен лежать там же где и xterm.
                В программе "putty" нужно в соотвецтвующей настройке указать, что она должна работать с файлом xterm-putty.
                По умолчанию там указан файл xterm.
                Этот файл нужен для того чтобы в программе "putty" правильно обрабатывались
                нажатия клавиш на клавиатуре. Программы системы iceB должны
                быть построены с параметром MAXFKEYS=10 (смотрите файл
                iceb.h в корне папки где находится система)
                
xterm-putty_12 - файл для программы "putty". Применяется если программы
		системы iceB построены с параметром MAXFKEYS=12 (смотрите файл
                iceb.h в корне папки где находится система). Проверена
		работа с этим файлом для debian 8

.bash_profile - этот настроечный файл нужно положить в домашний каталог каждого пользователя.
                Этот файл позволяет определить как пользователь вошёл в систему удалённо используя
                "putty" или локально. Если пользователь зашёл удалённо запустится скрипт "usermenu"
                и появиться меню для выбора дейсвий пользователем. Для того чтобы в меню была
                строка для запуска iceB пользователь должен входить в группу buhg.

usermenu      - скрипт создания меню для выбора действий опетатором после удалённог входа
                в систему. Должен лежать в папке /usr/local/bin

bankcp        - скрипт для передачи файлов по сети на машины работающие под Windows
                Должен лежать в папке /usr/local/bin
